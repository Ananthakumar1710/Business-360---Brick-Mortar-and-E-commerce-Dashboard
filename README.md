# Business-360---Brick-Mortar-and-E-commerce-Dashboard
📌 Project Overview
This project involves the creation of a comprehensive 360-degree Business Insight Dashboard for Atliq Hardware, a global electronics company. The dashboard provides data-driven insights across five critical departments to help stakeholders monitor performance, identify bottlenecks, and drive strategic growth in both Brick & Mortar and E-commerce channels.

🛠 Technologies Used
Business Intelligence: Power BI Desktop

Database Management: MySQL, SQL Server

Data Analysis & Optimization: DAX Studio, Excel

Languages: Power Query (M), DAX

🚀 Key Features & Impact
Multi-Departmental Analysis: Engineered a holistic view for Finance, Sales, Marketing, Supply Chain, and Executive teams.

Big Data Handling: Successfully integrated and transformed 1M+ records from hybrid sources (MySQL and Excel).

Performance Optimization: Utilized DAX Studio to refine data models, resulting in:

30% reduction in file storage size.

5% improvement in report loading performance.

Business Value: The insights derived from this dashboard championed a 10% increase in revenue and a 20% reduction in operational costs.

📊 Dashboard Views
Finance View: P&L statements, Net Sales trends, and Gross Margin percentages.

Sales View: Customer performance and market share analysis.

Marketing View: Product performance and segment-wise profitability.

Supply Chain View: Forecast accuracy and net error tracking to optimize inventory.

Executive View: High-level KPIs for market leadership and revenue contributions.

🛠 Data Modeling & DAX
The project follows a Star Schema for efficient data retrieval. Key DAX measures were created for:

Net Sales, Cost of Goods Sold (COGS), and Gross Margin.

Year-over-Year (YoY) Growth comparisons.

Target vs. Actual performance tracking.
